### 技术层面：

1.界面是通过 Java 内置的Swing 以及 Awt 实现的；💀

2.加密算法没有使用 Java 内置的算法包，而是手工实现的；💀

3.加密的方式是采用了字节流的读取，然后对读出来的内容进行加密；💀

4.本Demo是汲取网上的经验，通过自己集成以及更改完成的。💀
